To-Do List App

Project Description

The To-Do List App is designed to help users organize their tasks efficiently. Users will be able to add, edit, delete, and mark tasks as completed. The app will also support local storage to ensure tasks persist even after the page is refreshed.

Purpose

The main purpose of this project is to practice and apply HTML, CSS, and JavaScript skills in a collaborative development setting. This project will enhance our ability to work with local storage, DOM manipulation, and user-friendly interfaces.

Key Features

  • Add, edit, and delete tasks
  • Mark tasks as completed
  • Save tasks in local storage
  • Drag-and-drop sorting
  • Due date reminders
  • Dark mode toggle

Roles & Responsibilities

Repo Admin

Responsible for setting up and maintaining the GitHub repository, organizing folder structures, and managing permissions.

Secretary

Will take meeting minutes and store them in the repo. Responsibilities will rotate among team members.

Frontend Developers

Responsible for implementing the HTML and CSS layout, ensuring a clean and responsive design.

JavaScript Developers

Responsible for implementing task functionality, including adding, deleting, completing tasks, local storage, and drag-and-drop features.

Code Reviewers

Responsible for reviewing code changes, ensuring quality and consistency before merging to the main branch.
